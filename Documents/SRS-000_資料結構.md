# 資料結構

## 雅俗通字典

使用 SQLite 作為資料庫系統。

- 檔案名稱：Sip_Ngoo_Im.db
- 資料庫名稱：雅俗通十五音字典

### 聲母對照表

聲母對照表的 Table Schema 結構如下：

```sh
CREATE TABLE 聲母對照表 (
    識別號   INTEGER PRIMARY KEY NOT NULL UNIQUE,
    聲母碼   TEXT    NOT NULL,
    國際音標 TEXT,
    白話字   TEXT,
    台羅     TEXT,
    閩拼     TEXT,
    方音     TEXT,
    十五音   TEXT,
    備註     TEXT
);
```
