# 閩南話漢字標音轉換指引 (Southern Min Phonetic Conversion Guide)

## 版本：V1.0

這份文件定義了本系統以 **TLPA+** （台語音標）為核心中繼格式（Pivot Format）的轉換標準，確保不同標音系統間的轉換精確且美觀。

【註】：為求本文件之簡潔、易讀，以下若見 **TLPA** ，其義等同： TLPA+ 。

---

## 1. 核心標準：TLPA (中繼格式)

系統內部統一使用 **TLPA** 作為處理標準。其核心特色在於**聲母單字母化**（所有的【聲母】皆只用一個羅馬拼音字母表示），故而有助於程式編碼，自【音節】中切分出【聲母】與【韻母】及不同標音系統間之轉換：

- **z** (取代 ts / ch)
- **c** (取代 tsh / chh)
- **s / j** (保持不變)
- **ø** (代表【零聲母】；但基本上都怱略不標示)

---

## 2. 各標音系統轉換執行重點

### A. 白話字 (Pe̍h-ōe-jī, POJ)

- **聲母轉換**：`z` → `ch`, `c` → `chh`。
- **韻母變更**：
  以下為【韻母】在 TLPA 轉換成 POJ 的規則：
  - `ue` → `oe`
  - `ua` → `oa`
  - `ik` → `ek`
  - `ing` → `eng`
  - `oo` → `o\u0358` (o加上點 `o͘`)
  - `nn` → `\u207F` (鼻音化小 `ⁿ`)
  - `nnh` → `h\u207F` (鼻音化小 `hⁿ`)
- **調符位置**：遵循優先級規則 (a > o > e > i > u)。
  - 標記之調號為該對應漢字之本調，至於實際讀出的聲調則需以連續變調之規則進行變調。外來語則擬似本調。
  - 當一個音節有多個字母時，調號得標示在響度最大的字母上面（通常在韻腹）。響度順序： a > oo > (e = o) > (i = u)
  - 雙母音 iu 及 ui ，調號都標在後一個字母上，因為前一個字母是介音。
  - m 作韻腹時標於字母 m 上。 oo 及 ng，標於前一個字母上；比如 ng 標示在字母 n 上。
  - 三合字母 ere，標於最後的字母 e 上。

### B. 台羅拼音 (Tâi-lô)

- **聲母轉換**：`z` → `ts`, `c` → `tsh`。
- **韻母變更**：保持 TLPA 原始結構（如 `ue`, `ua`, `ik`, `ing` 均不變）。
- **調符位置**：遵循優先級規則。

### C. 閩拼方案 (Bân-phing, BP)

- **零聲母處理**（變形規則）：
  - 以 `i` 開頭：若後接韻母則 `i` 變 `y`，若僅有 `i` 則加 `y` 前綴。
  - 以 `u` 開頭：若後接韻母則 `u` 變 `w`，若僅有 `u` 則加 `w` 前綴。
- **韻母變更**：
  以下為【韻母】在 TLPA 轉換成 BP 的規則：
  - `au` → `ao`
  - `iau` → `iao`
- **鼻化韻母變更**：
  BP 採【前置】方式標注鼻化韻母：將【羅馬拼音字母】：**n** 標注於【韻母】之前；不同於【白話字】的【後置】方式 `ⁿ`。
  `【韻母】nn` → `n【韻母】`
  - `ann` → `na`
  - `ainn` → `nai`
  - `aunn` → `nao` （韻母亦需轉換）
  - `enn` → `ne`
  - `inn` → `ni`
  - `iann` → `nia`
  - `iaunn` → `niao` （韻母亦需轉換）
  - `ionn` → `nioo` （TLPA 的【韻母】：ioo **[iɔ]**，鼻化韻本該寫作： **ioonn**，但簡化成：**ionn**，故在轉換成 BP 時，韻母需改回原有的：**ioo**，再轉成**nioo**）
  - `onn` → `noo`（TLPA 的【韻母】：oo **[ɔ]**，鼻化韻本該寫作： **oonn**，但簡化成：**onn**，故在轉換成 BP 時，韻母需改回原有的：**oo**，再轉成**noo**）
  - `unn` → `nu`
  - `uainn` → `nuai`
  - `uenn` → `nue`
  - `uinn` → `nui`
- **調號映射**：BP 的調號定義與台羅/白話字不同（例如台羅 5 聲在 BP 可能映射為 2 聲，需查表）。

### D. 方音符號 (TPS)

- **聲母轉換**：
  - 雙唇音（清聲送氣塞音/清聲不送氣塞音/濁聲不送氣塞音/濁聲鼻音）：
    - ph → ㄆ [ph]
    - p → ㄅ [p]
    - b → ㆠ [b]
    - m → ㄇ [m] (鼻音)

  - 軟顎音（清聲送氣塞音/清聲不送氣塞音/濁聲不送氣塞音/濁聲鼻音）：
    - kh → ㄎ [kh]
    - k → ㄍ [k]
    - g → ㆣ [k]
    - ng → ㄫ [ŋ] (鼻音)

  - 齒齦音（清聲送氣塞音/清聲不送氣塞音/濁聲邊音/濁聲鼻音）：
    - th → ㄊ [th]
    - t → ㄉ [t]
    - l → ㄌ [l]（邊音）
    - n → ㄋ [n] (鼻音)

  - 齒齦音（清聲送氣塞擦音/清聲不送氣塞擦音/濁聲不送氣塞擦音/清聲擦音）：
    - c → ㄘ [tsh]
    - z → ㄗ [ts]
    - j → ㄌ [d͡z]
    - s → ㄙ [s] (擦音)

  - 齦顎音（清聲不送氣塞擦音/清聲送氣塞擦音/濁聲不送氣塞擦音/清聲擦音）：
    - ci → ㄑㄧ [t͡ɕʰ]
    - zi → ㄐㄧ [t͡ɕ]
    - ji → ㆢㄧ [d͡ʑ]
    - si → ㄒㄧ [ɕ] (擦音)

    【齒齦音/舌齒音】聲母，後接【韻母】：**i** 時（或簡稱：**介音顎化**）：當聲母為 `z, c, s, j` ，且其後接 `i` 系列韻母時，需轉換為顎化符號（`ㄐ, ㄑ, ㄒ, ㆢ`）。
    【齒齦音 + i】轉換成【齦顎音 + i 】規則：
    - j+i: ㆡ[j] + ㄧ[i] = ㆢ[J] + ㄧ[i]
    - z+i: ㄗ[z] + ㄧ[i] = ㄐ[Z] + ㄧ[i]
    - c+i: ㄘ[c] + ㄧ[i] = ㄑ[C] + ㄧ[i]
    - s+i: ㄙ[s] + ㄧ[i] = ㄒ[S] + ㄧ[i]

  - 聲母音
    - h → ㄏ [h] (清聲擦音)
    - (零聲母) → （無對映之方音符號，故不標示） [ʔ]

- **韻化輔音變更**：
  韻化輔音指**亦可當作【韻母】使用的【聲母】（亦稱：輔音）**
  - m → ㆬ
  - ng → ㆭ

- **韻母變更**：
  - a → ㄚ
  - i → ㄧ
  - u → ㄨ
  - e → ㆤ
  - oo → ㆦ [ɔ]
  - o → ㄜ [ə]
  - ai → ㄞ（不要轉成：ㄚㄧ）
  - au → ㄠ（不要轉成：ㄚㄨ）
  - am → ㆰ（不要轉成：ㄚㆬ）
  - ang → ㄤ（不要轉成：ㄚㆭ）
  - an → ㄢ（不要轉成：ㄚㄣ）
  - om → ㆱ（不要轉成：ㆦㆬ）
  - ong → ㆲ（不要轉成：ㆦㆭ）
  - ing → ㄧㄥ [iəŋ]（不要轉成：`ㄧㆭ`）

    **【註】：其它注意要點**
    - 韻尾 `-n`，轉成 `ㄣ` ，但只用在結合韻母：ㄧㄣ、ㄨㄣ及ㆨㄣ。

- **鼻化韻母變更**：
  - ainn → ㆮ
  - aunn → ㆯ
  - ann → ㆩ
  - inn → ㆪ
  - unn → ㆫ
  - enn → ㆥ
  - onn → ㆧ

- **入聲韻尾變更**：
  - `-p(4/8)` → `ㆴ / ㆴ˙`
  - `-t(4/8)` → `ㆵ / ㆵ˙`
  - `-k(4/8)` → `ㆻ / ㆻ˙`
  - `-h(4/8)` → `ㆷ / ㆷ˙`

- **聲調**：
  - 1 聲（陰平調）：（無調符）
  - 2 聲（陰上調）： ˋ（U+02CB）
  - 3 聲（陰去調）： ˪（U+02EA）
  - 4 聲（陰入調）：（無調符），如：ㆴ、ㆵ、ㆻ、ㆷ
  - 5 聲（陽平調）： ˊ（U+02CA）
  - 6 聲（陽上調）：（無調符）
  - 7 聲（陽去調）： ˫（U+02EB）
  - 8 聲（陽入調）： ˙（U+02D9），如：ㆴ̇、ㆵ̇、ㆻ̇、ㆷ̇

- **調符位置**：顯示於漢字右側（使用 `<rtc>` 標籤）。

#### 【補充】

- **聲母與發聲部位對照**
  - 唇音/雙唇音: p, ph, b, m ( BP: b, p, bb, m )
  - 舌音/舌尖中音（齒齦音）: t, th, n, l (BP: d, t, ln, l)
  - 齒音/舌尖前音（齒齦音）: ts/z, tsh/c, j, s ( BP: z, c, j, s )
  - 齒音/舌尖前音（齦顎音）: zi, ci, ji, si ( BP:j, ch, jj, sh )
  - 牙音/舌根音（軟顎音）: k, kh, g, ng ( BP: k, kh, g, ng )
  - 喉音（聲門音）：h ( BP: h )

- **查表法**：依據 `phonetic_mapping.json` 進行聲母、韻母、調符的一對一映射。

## 3. 調符套用邏輯 (Tone Mark Logic)

### 優先級規則 (Placement Priority)

當一個音節有多個母音時，調符應標註於權重最高的字母上：

1.  **a** (最高)
2.  **oo / o**
3.  **e**
4.  **i / u** (後者優先，如 `ui` 標在 `i`，`iu` 標在 `u`)

### 組合字元 (Combining Diacritics)

- **第 8 聲 (陽入調)**：必須使用 `\u030D` (Combining Vertical Line Above)。
- **標準化**：輸出前應執行 `normalize("NFD")`，這能讓字型（如 Charis SIL）更好地處理標記定位。

## 4. 關鍵技術修復與美學規範

以下是本系統開發過程中發現的關鍵技術要領，對標音穩定度至關重要：

### I. 零聲母對齊修復 (The ZWSP Hack)

- **問題**：瀏覽器在渲染以母音開頭（零聲母）且帶有組合調符（如 `e̍k`）的字串時，常會發生對齊位移。
- **解決方案**：在所有標音字串前端注入 **`\u200B` (零寬度空格)**。這能為調符提供一個穩定的錨點，確保對齊完美。

### II. 靜態 HTML 結構緊湊化

- **規範**：生成的 HTML 源碼中，`<ruby>`、`<rt>`、`<rtc>` 標籤之間**嚴禁換行或空格**。
- **理由**：HTML 內的任何空白字元在瀏覽器中都會被轉換為可視空格，導致漢字間距被拉開，特別是在手機 RWD 模式下會造成排版崩潰。

### III. 響應式字體縮放 (RWD Title)

- **規範**：在小螢幕（< 600px）下，標題字體應自動降級至 `1.3rem ~ 1.5rem`，並強制 `white-space: nowrap`。
- **理由**：標題（如《寒窯賦》）若因螢幕太窄而折行，會嚴重影響導讀體驗。

### IV. 字型依賴 (Font Sovereignty)

- **規範**：系統必須內建並載入 `Charis SIL` 字型。
- **功能需求**：CSS 必須啟用 `font-feature-settings: "ccmp" 1, "mark" 1, "mkmk" 1;`。
- **理由**：這是目前唯一能保證台羅/白話字調符不位移、不重疊的工業級解決方案。
