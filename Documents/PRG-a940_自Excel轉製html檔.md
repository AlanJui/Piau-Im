# a940\_自Excel轉製html檔.py

## 功能說明

這個程式會讀取當前 作用中 (Active) 的 Excel 工作表，並依據 A column (漢字) 與 B column (漢字標音) 的內容，產生對應的 HTML 檔案。

## 邏輯規則：

1. 資料讀取：從第 2 列開始讀取 A 與 B 欄，直到最後一列。

1. HTML 結構：
   - 若 B 欄有內容 (漢字標音)，產出 <ruby>A欄內容<rt>B欄內容</rt></ruby>。
   - 若 B 欄為空，則直接產出 A 欄內容 (純文字/標點符號)，不包覆 ruby 標籤。

1. 段落控制：
   當 A 欄內容為換行符號 (\n 或 Excel 的 CHAR(10)) 時，程式會自動結束目前的段落 </p> 並開始新的段落 <p>。

1. **樣式**：已預設簡單的 CSS，包含字型 (微軟正黑體)、行高與 ruby 標籤樣式。

## 使用方式

1. 確認您要轉換的 Excel 檔案已經開啟，且游標停留在正確的工作表上。

2. 在終端機執行以下指令 (可指定輸出檔名，若不指定預設為 ./docs/output_from_excel.html)：

```bash
python a940_自Excel轉製html檔.py 您的輸出檔名.html
```

【註】：**目錄自動建立**：程式執行時會檢查輸出路徑的父目錄 (即 docs) 是否存在，若不存在會自動建立，避免發生錯誤。
