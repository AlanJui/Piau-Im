# a999\_自動生成index_html

## 功能摘要

- 將 docs 目錄下的所有 .html 檔案，製作成一目錄索引，置入 docs/index.html 檔案中。
- 提供【漢字標音切換】功能，透過 JavaScript，令使用者可在每個【漢字標音網頁】中，切換【漢字】之【標音格式】，如：【反切】、【注音】、【拼音】等。

## 撰碼說明

### 台語音標與十五音互轉

參考 Rime 輸入方案，「十五音輸入法【台語音標拼音】（sgi_ping_im.schema.yaml）」。

- **algebra：拼寫運算規則（字典/輸入碼對映規範）**: 指示 Rime 平台，對於【輸入方案】使用之【字典】（亦稱：詞典），於讀入之後，
  如何解析各【漢字】之【台語音標】所形成之【音節】，將音節中之【羅馬拼音字母】，轉換成【輸入方案】之【輸入碼】。
  上述之解析，構成【輸入碼對照表】，可供【輸入方案】之【輸入方案之翻譯器】（以下簡稱：【翻譯器】）作為：處理鍵盤按鍵之作業依據。
- **translator/preedit_format（輸入翻譯器/預編輯格式）**：指示【輸入翻譯器】，對於使用者在盤鍵之【按鍵】，如何依據【輸入碼對照表】
  將【按鍵】轉成【輸入碼】，以便【輸入翻譯器】可依【輸入碼】從【字典】中查字；同時亦可將【按鍵】輸出，於【輸入編碼列】中顯示。
- **translator/comment_format（輸入翻譯器/註解格式）**：指示【輸入翻譯器】如何將【按鍵】已轉換成的【輸入法】，依據【輸入碼對照表】
  轉換成【候選字清單】中可供選取之同音節漢字；且於各漢字後，附上【漢字標音】音節（反切/注音/拼音）。

### 預設應有之漢字標音格式

參考**a410\_批次式漢字標音網頁製作.py（Line: 192 - 228）**，規範常用之【漢字標音】格式之名稱，定義每個【漢字】之【上方】、【右方】
應使用何稱【漢字標音】標準，如：【反切】使用【十五音】；【注音】使用【方音符號】；【拼音】則有【閩拼調符】、【閩拼調號】、【台語音標】、

## 配合作業

### 製作漢字標音網頁

為配合【漢字拼音切換】功能，原【a400*製作標音網頁.py】與【a410*批次式漢字標音網頁製作.py】
兩程式需配合變更，於<ruby>標籤中附上 data-tlpa attribute ，加入【漢字】之【台語音標】音節。

### 導覽操作

自【首頁】進入每個漢字標音網頁後，網頁之【頂端】與【底端】，應有【導覽列】。

【導覽列】需有：【回首頁】、【方音符號】、【閩拼方案】、...【台語音標+十五音】、【自訂】：《漢字上方下拉選單》、《漢字右方下拉選單》按鈕。

是否【漢字標音格式】的切換，亦可用【上一頁】、【下一頁】按鈕來操作。
