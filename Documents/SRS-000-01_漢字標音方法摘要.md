# 漢字標音方法摘要

## 漢字標音結構

漢字標音 = 聲母 + 韻母 + 聲調

閩南話之【聲調】表示，大致有兩種：（1）聲調符號（調符）；（2）聲調數值（調號）。

---

## 台羅音標(TL)

台羅音標(TL)亦稱作【台羅拼音】。 由教育部發佈，作為【台語羅馬字母拚音】標準。

台羅音標預設之【聲調】表示法，採用：調符，如：【台】字的讀音標注為：Tâi 。本標準在制定之初，為顧及當時電腦操作之便，故亦允許將【調符】改成【調號】，置於拼音（音節）之末，如：Tai5，此種【標音格式】亦稱作：【輸入式】；為書寫之便，允許【調號】為 1（陰平調）、4（陰入調）可略去不寫。

Tâi = T + âi
    = 聲母 + 標調韻母
    = 聲母 +（韻母 + 聲調符號）
    = T + ai + 5

因【台羅音標】為台灣區，閩南話之強勢拼音系統，網路線上的【漢字讀音字典】，多數以【台羅音標】為主，為方便操作，故而允許填寫【台語音標】時，可直接複製並貼上取自線上字典的【台羅音標】。

---

## 台語音標(TLPA)

台語音標(TLPA)，為【台羅音標】之前身。

本專案採用【台語音標】作為【羅馬拼音字母】的標準，英文簡寫：TLPA；是為【台羅音標】/【台羅拼音】的前身。

TLPA【台語音標】的特色為：讀音=聲母+韻母+調值（聲調），如：【台】字的讀音=t + ai + 5，此種讀音之標示法，便於程式撰寫，易於切分或組合：聲母、韻母、聲調。而【台羅音標】：Tâi = T + âi = 聲母+標調韻母=聲母+（韻母+聲調符號），欲得【聲調】，得自標調韻母【âi】抽離，取得5。

### 台語音標與台羅音標相容性

簡單說【台羅音標】是【台語音標】延伸改版，故而兩者之間幾無差異。

聲母差異：

| 台羅音標 | 台語音標 | TLPA+ |
|----|---|----|
| ts | c | z |
| tsh | ch | c |

### 聲調格式

【台語音標】TLPA 拼音系統，其【聲調】之【格式】亦有兩種，本專案以【調號式】為標準。：

以漢字【泉】為例：

- 調符式（帶聲調符號）：【zuânn】
- 調號式（無聲調符號）：【zuann5】

### 調值/調符/調名對映

以字母 i 舉例說明【聲調符號】：

1. "i": 陰平調
2. "í": 陰上調
3. "ì": 陰去調
4. "i": 陰入調
5. "î": 陽平調
6. "ǐ": 陽上調
7. "ī": 陽去調
8. "i̍": 陽入調

### 轉換規則

1. 拼音完全沒有聲調符號，如：約【ioh】、因【in】
   - 如果拼音字母尾端為字母：h/p/t/k，表示：音標為【陰入調】，調值為：4，如：【約】【ioh=ioh4】；
   - 非上狀況者，表示：音標為【陰平調】，調值為：1，如：【因】【thinn=in1】泉】的 TLPA 拼音為【 tsuânn】在字母 a 上的【聲調符號】抽掉，改成【聲調數值】：5，最後將拼音重組成【tsuann5】。

## 【台語標音】改良版（TLPA+）簡介

台語音標改良版幾乎等同原版，不同處僅兩點：

|序號|台語音標原版(TLPA)|台語音標改良版(TLPA+)|
|----|---------|-----------|
| 1 |c  | z |
| 2 |ch | c |

簡單解說【TLPA+】：它令標音系統中使用【聲母】完成單一字母化，不會出現如：ch，使用兩個拼音字母之狀況；
這對閱讀之視覺感受及程式編碼控制均較優。

# 方音符號

參考【注音符號】，依閩南話發音需求，改良而成之【注音】方法，簡稱：方音符號。

【方音符號】之構成，亦為：【聲母】、【韻母】、【調符】。

如：

 1. 歸：ㄍㄨㄧ
    - 聲母：ㄍ
    - 韻母：ㄨㄧ
    - 調符：1

 2. 己：ㄧˋ
    - 聲母：
    - 韻母：ㄧ
    - 調符：2

 3. 敬：ㄍㄧㄥ˪
    - 聲母：ㄍ
    - 韻母：ㄧㄥ
    - 調符：3

 4. 覺：ㄍㄚㆻ
    - 聲母：ㄍ
    - 韻母：ㄚㆻ
    - 調符：4

 5. 來：ㄌㄞˊ
    - 聲母：ㄌ
    - 韻母：ㄞ
    - 調符：5

 7. 是：ㄒㄧ˫
    - 聲母：ㄒ
    - 韻母：ㄧ
    - 調符：7

 8. 獨：ㄉㄚㆻ˙
    - 聲母：ㄉ
    - 韻母：ㄚㆻ
    - 調符：7


### 方音符號解析流程

 1. 檢查輸入之【方音符號】的最尾一個字元是否為【方音調符】。若有，對映成【調號】。

    檢查用之 pattern 。

    ```python
    HongImTiauHu = re.compile(r"ˋ|˪|ˊ|˫|\u02D9", re.I)
    ```

    透過 Dictionary 對映，將【方音調號】對映成【調號】。

    ```python
    Hong_Im_Tiau_Hu_Dict = {
        "ˋ"    : 2,
        "˪"     : 3,
        "ˊ"    : 5,
        "˫"     : 7,
        "\u02D9": 8,
    }
    ```

 2. 判斷沒有【方音調符】的【方音符號】，是第一聲？還是第四聲？

    若是【方音符號】最尾端的字元為：ㆴ、ㆵ、ㆻ、ㆷ，則表示為第四聲（陰入）；
    若否，則為第一聲（陰平）。

 3. 將【方音符號】剔除【方音調符】成為：【無調符方音符號】。

    如：

    - 【方音符號】：ㄒㄧ˫   ==> 【無調符方音符號】： ㄒㄧ
    - 【方音符號】：ㄉㄧㆲ˪ ==> 【無調符方音符號】： ㄉㄧㆲ

 4. 將【無調符方音符號】的第一個字元抽出當【聲母】；餘下的【無調符方音符號】
    當作【韻母】。
